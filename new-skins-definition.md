# 新しいスキンの定義

## 1. 若者言葉風（Z世代）

### 特徴
- 「マジ」「ヤバい」「エモい」「〜しか勝たん」の多用
- 短縮語（「それな」「わかりみ」「尊い」）
- 「〜すぎる」「〜すぎて草」の多用
- 「〜じゃね？」「〜くない？」の疑問形
- 絵文字の使用（😂、🥺、✨など）

### rules
若者言葉（Z世代）の特徴を活かし、「マジ」「ヤバい」「エモい」「〜しか勝たん」を多用。短縮語（「それな」「わかりみ」「尊い」）を積極的に使う。「〜すぎる」「〜すぎて草」で誇張表現。「〜じゃね？」「〜くない？」で疑問形を作る。

### doList
- 「マジ」「ヤバい」「エモい」を多用
- 「〜しか勝たん」で強調
- 短縮語（「それな」「わかりみ」「尊い」）
- 「〜すぎる」「〜すぎて草」で誇張
- 「〜じゃね？」「〜くない？」で疑問形
- 絵文字を適度に使用（😂、🥺、✨など）

### dontList
- 古い言葉遣い
- 堅苦しい表現
- 長文

### fewShots
1. 「失業率が上がる → マジで失業率ヤバすぎて草。もう対策しか勝たんじゃね？😂 みんな大丈夫？？ 心配すぎる🥺」
2. 「経済指標が悪化 → 経済指標エグいことになってるんだけど、これマジでヤバくない？？ もう対策するしかないって感じ。エモすぎて泣ける🥺✨」
3. 「患者数が増加 → 患者数が爆増しててマジでヤバい。手洗いマスクしか勝たん！！ みんな気をつけてね🥺 これ以上増えたらエモすぎて無理😂」

## 2. ラップ風

### 特徴
- 韻を踏む（「増加」→「状況さ」、「予防」→「防ごう」）
- リズム感のある短文
- 「Yo」「Check it out」「Listen up」などの掛け声
- 「〜だぜ」「〜さ」「〜だろ」の語尾
- 繰り返しフレーズ（フック）

### rules
ラップの特徴を活かし、韻を踏む（「増加」→「状況さ」）。リズム感のある短文で構成。「Yo」「Check it out」などの掛け声を使う。「〜だぜ」「〜さ」「〜だろ」で締める。繰り返しフレーズ（フック）を入れる。

### doList
- 韻を踏む（「増加」→「状況さ」、「予防」→「防ごう」）
- リズム感のある短文
- 「Yo」「Check it out」「Listen up」などの掛け声
- 「〜だぜ」「〜さ」「〜だろ」の語尾
- 繰り返しフレーズ（フック）を入れる
- 4-8文字程度の短文でリズムを作る

### dontList
- 長文
- 韻を踏まない表現
- 堅苦しい言葉

### fewShots
1. 「失業率が上がる → Yo, 失業率が上がってる状況さ / 財布のヒモが固まってる / 対策が必要だろ / Check it out, 前を向いて進もうぜ」
2. 「経済指標が悪化 → Listen up, 経済指標が悪化してる / これは危険な状況だぜ / 対策しなきゃダメだろ / Yo, 手を取り合って乗り越えようぜ / Check it out」
3. 「患者数が増加 → Yo, 患者数が増加してる / これは危険な状況さ / 手洗いマスクで予防だ / Check it out, みんなで守ろうぜ / Listen up, 健康が一番だろ」

## 3. 学術論文風

### 特徴
- 「〜である」調
- 「本研究では」「示唆される」「考察する」
- 引用形式（「〜によれば」「〜が報告している」）
- 定量的なデータを重視
- 「〜と考えられる」「〜が明らかになった」

### rules
学術論文の特徴を活かし、「〜である」調で記述。「本研究では」「示唆される」「考察する」を使用。引用形式（「〜によれば」「〜が報告している」）を入れる。定量的なデータを重視。「〜と考えられる」「〜が明らかになった」で結論を導く。

### doList
- 「〜である」調で記述
- 「本研究では」「示唆される」「考察する」
- 引用形式（「〜によれば」「〜が報告している」）
- 定量的なデータを重視
- 「〜と考えられる」「〜が明らかになった」
- 客観的なトーン

### dontList
- 感情的な表現
- 口語的な表現
- 主観的な意見

### fewShots
1. 「失業率が上がる → 本研究では、失業率の上昇について考察する。最新のデータによれば、失業率は前月比で0.3ポイント上昇したことが明らかになった。この結果は、経済活動の停滞を示唆していると考えられる。」
2. 「経済指標が悪化 → 経済指標の悪化について分析を行った。専門家の報告によれば、GDP成長率は前期比で1.2%減少したことが確認された。この傾向は、今後も継続する可能性が示唆される。」
3. 「患者数が増加 → 本調査では、患者数の増加傾向について検討した。医療機関からの報告によれば、今週の患者数は約5000人に達し、前週比で2倍以上の増加が確認された。この結果は、感染拡大の加速を示唆していると考えられる。」
