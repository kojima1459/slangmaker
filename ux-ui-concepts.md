# UX/UI心理学コンセプト集

PDFが空白のため、一般的なUX/UI心理学の原則を適用します：

## 適用する原則

1. **ヒックの法則** - 選択肢が多いほど決定に時間がかかる
   - スキン選択をシンプルに、カテゴリ分けで認知負荷を軽減

2. **フィッツの法則** - ターゲットが大きく近いほどクリックしやすい
   - CTAボタンを大きく、重要な要素を目立たせる

3. **ミラーの法則** - 人は一度に7±2個の情報しか保持できない
   - 情報を適切にグループ化、階層化

4. **視覚的階層** - 重要度に応じて要素のサイズ、色、配置を調整
   - タイポグラフィで明確な階層を作る

5. **余白の美学** - 適切な余白で読みやすさと理解度を向上
   - カードやセクション間に十分なスペース

6. **一貫性の原則** - 同じ要素は同じスタイルで統一
   - ボタン、カード、フォントを統一

7. **フィードバックの原則** - ユーザーの行動に対して即座にフィードバック
   - ホバー効果、ローディング状態、成功メッセージ

8. **プログレッシブディスクロージャー** - 必要な情報だけを段階的に表示
   - 詳細設定を折りたたみ、チュートリアルをステップ化
